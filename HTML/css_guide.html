css 가이드

css 작성 규칙

  • 영문 소문자로만 작성한다.
  • 들여쓰기를 하지 않는다.
  • 불필요한 공백은 최소화 한다.
  • 코드 그룹 사이에 빈 줄을 넣을 수 있으나 한줄을 넘지 않게 한다. (빈줄은 선택사항)
  • 선택자와 속성, 속성값 사이에 줄바꿈을 하지 않는다. (한줄로 정의)
  • 속성값 축약 (컬러 코드값, 위치값, 0단위)
  • 큰따옴표를 사용한다.
