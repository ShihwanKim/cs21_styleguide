table태그

DTD에 따라 사용되는 요소의 차이가 있다

  • html5일 경우 : summary 필요없음, caption에 summary 내용 추가
  • html5 이외 : summary 필수(표의 간략한 설명), caption 필수 (표 요약)
  • tbody 의 td에 셀병합이 있는 경우 id와 headers로 매칭시킨다

스크롤들어가는 컨텐츠

div태그에 스크롤들어가는 컨텐츠 (회원약관) 에는 tabindex=”0”을 넣어준다.
스크린리더의 포커스가 div에는 잡히지 않기 때문에 div안에 내용을 읽을 수 있도록 tabindex=”0”으로 스크린리더의 포커스를 잡아준다.
